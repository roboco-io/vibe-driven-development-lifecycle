# テンプレート

VDLC六つのステージで繰り返し必要になる文書形式を五種類に整理した。空の様式ではなく、記入ガイドと例文の断片を含む実戦型テンプレートなので、各ページからそのままコピーして埋めて使う。

| テンプレート | いつ使うか | 関連ガイドステージ |
|---|---|---|
| [意図文書](/ja/templates/intent-doc) | 何を、なぜ作るのかを6-pager縮約形で合意し文書に残すとき | [ステージ1 — 意図定義](/ja/guide/intent) |
| [PR-FAQ](/ja/templates/pr-faq) | 完成した姿をリリース発表文と想定FAQで先に描いてみるとき | [ステージ1 — 意図定義](/ja/guide/intent) |
| [リスクマトリクス](/ja/templates/risk-matrix) | 成果物のリスク等級をつけ、検証強度を定めるとき | [ステージ4 — 検証](/ja/guide/verify) |
| [レビューチェックリスト](/ja/templates/review-checklist) | エージェントの成果物を人または交差レビューのエージェントが検討するとき | [ステージ4 — 検証](/ja/guide/verify) |
| [CLAUDE.md例](/ja/templates/claude-md) | プロジェクトルールファイルの骨格を立てる、または既存ファイルを点検・更新するとき | [ステージ2 — コンテキスト設計](/ja/guide/context) |
